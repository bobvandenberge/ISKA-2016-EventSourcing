Wanneer EventSourcing?

  • Rich object-oriented domain models
  • Non-CRUD behaviour
  • Good debugging capabilities is critical
  • Need version control of data (auditing)
  • Need it for monitoring
  • Domain is inherently event driven
  • You are using CQRS
